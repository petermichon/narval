# Narval

## Installation

Clone the repository

```bash
git clone https://github.com/petermichon/narval.git narval &&
cd narval/src/
```

Install dependencies

```bash
deno run install
```

Build the website and copy it to the server

```bash
deno run build
```

Serve the website over localhost (https://localhost:8443/)

```bash
deno run serve
```
